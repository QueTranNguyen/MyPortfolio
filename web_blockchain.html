Final Project - Unleashing the Power of Blockchain in Smart Supply Chains30 November 2024

The Vaccine Chain project is a vaccine storage temperature monitoring system that ensures safety and transparency in the vaccine supply chain by utilizing IoT and blockchain technology. The system monitors temperature during vaccine transportation with ~90% accuracy while storing data on the blockchain to ensure integrity, transparency, and immutability.

The project emphasizes technological innovation, integrating IoT, blockchain, and data management for secure, transparent vaccine monitoring, prioritizing a scalable architecture over extensive user features.

1. The system is designed with two key roles:

Admin
  • Manage Devices, Vaccines, and Doses.
  • Perform Device-Vaccine Connections.
  • View detailed Logs of recorded data.
  • Access and analyze system Statistics.
User
  • Search for vaccine information by ID.
  • Access detailed vaccine logs for enhanced transparency and tracking.

2. System Workflow:

Step 1: Data Collection

- IoT sensors mounted on ESP32 microcontrollers collect real-time temperature data via WiFi.

  • Device and vaccine temperatures
  • Sensor type
Step 2: Data Processing and Storage

- Collected data is processed and stored in two locations:

  • On-chain (Blockchain): Stores critical monitoring data (e.g., IDs and temperatures) using smart contracts for tamper-proof security.
  • Off-chain: Stores comprehensive vaccine and device details for efficient data management.
Step 3: Data Retrieval and Integration.

- The system retrieves and integrates data for user queries:

  • Off-chain: Provides detailed vaccine information (e.g., expiration, manufacturer, administrator).
  • On-chain: Fetches blockchain transactions for a complete vaccine history.
